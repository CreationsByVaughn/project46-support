Project 4:6 – Support

Thank you for using Project 4:6, a space for prayer, reflection, and spiritual growth. We're here to help if you have any questions or issues using the app.

Common Questions

  • How do I add a new prayer? → Tap the plus (+) button at the bottom of the screen.
  • Can I add a Bible verse to a prayer? → Yes! Each prayer entry has an optional verse field.
  • How do I set reminders? → Tap the reminder icon when adding or editing a prayer.
  • What happens after the 7-day trial? → You’ll be prompted to unlock the app with a one-time purchase to continue using all features.
